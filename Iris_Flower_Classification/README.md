# Iris Flower Classification

## Project Overview
This project focuses on classifying iris flowers into different species using machine learning techniques. It uses features like sepal length, sepal width, petal length, and petal width to predict the flower category.

## Objective
To build a classification model that accurately predicts the species of an iris flower based on its physical features.

## Dataset
The dataset contains the following features:
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  
- Species (Target Variable)

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

## Model Used
- Logistic Regression / Classification Algorithm

## Workflow
1. Data loading  
2. Data preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Model training  
5. Model evaluation  

## Result
The model successfully classifies iris flowers into their respective species with good accuracy.
