# CodeAlpha Machine Learning Projects

This repository contains 3 machine learning projects completed as part of the CodeAlpha internship.

## Projects Included

### Iris Flower Classification
- Classification model to predict iris flower species
- Uses dataset features and ML classification techniques

### Car Price Prediction
- Predicts car prices using Linear Regression
- Includes data preprocessing and visualization

### Unemployment Analysis
- Data analysis project on unemployment dataset
- Visualizations used to understand trends

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib
